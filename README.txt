In development. When you run it:

Uses Chrome's current Google account.
Creates alerts specified by DesiredKeywords variable.
Reads and displays all existing RSS feed entries for the alerts.

# creates alerts from DesiredKeywords, displays all RSS entries
npm run sync

# same, but in debug mdoe
npm run debug
